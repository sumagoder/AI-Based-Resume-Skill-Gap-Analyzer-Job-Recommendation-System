# AI-Based Resume Skill Gap Analyzer & Job Recommendation System

## Why This Project?

In today’s competitive job market, many students and job seekers struggle to identify the exact skills required for specific job roles. Traditional resume screening methods are manual, time-consuming, and often rely only on keyword matching.

This project was developed to help candidates analyze their resumes, identify missing skills, improve ATS performance, and receive suitable job recommendations using AI, Machine Learning, and NLP techniques.

---

## Project Overview

The AI-Based Resume Skill Gap Analyzer is an intelligent system that analyzes uploaded resumes and compares candidate skills with industry job requirements.

The system:

* Extracts resume content automatically
* Identifies technical and soft skills
* Detects skill gaps
* Predicts suitable job roles
* Generates ATS Resume Score
* Provides learning recommendations

The platform supports PDF, DOCX, and image-based resumes using OCR.

## Working of the System

1. User uploads resume
2. Resume text is extracted
3. NLP preprocessing is applied
4. Skills are extracted automatically
5. Resume is compared with job dataset
6. Skill gaps are identified
7. ATS score is generated
8. Suitable job roles are recommended
